# Emoji Images

This directory contains character images for PJSK emoji generation.

## Image Requirements
- Format: PNG with transparency
- Recommended size: 400x400px
- Background: Transparent
- Style: Consistent art style matching the game

## File Naming Convention
Images should be named using the character ID:
- miku.png (初音未来)
- ichika.png (星乃一歌)
- saki.png (天马咲希)
- honami.png (望月穗波)
- shiho.png (日野森志步)
- akito.png (东云彰人)
- toya.png (青柳冬弥)
- kohane.png (小豆泽心羽)
